How To Use NoteIT
Home Icon: Takes you to Home Page

Save Icon: Saves the note | Notes will auto save when clicking out of the pop up
Download Icon: Downloads a .txt file of all notes if clicked from home/settings page, otherwise only the current note is downloaded
